## 📝 案例详情

### 案例1：智能客服RAG系统

**场景**：电商客户服务

**功能**：
- 回答常见问题（FAQ）
- 订单查询
- 产品推荐
- 多轮对话

**技术点**：
- 基础RAG流程
- Streamlit界面
- 向量数据库
- 对话历史管理

**适合**：RAG入门学习者

**预计时间**：2-3小时

---

### 案例2：技术文档问答系统

**场景**：技术团队文档检索

**功能**：
- 快速搜索API文档
- 代码示例检索
- 多文档联合查询
- 相关性排序

**技术点**：
- 混合检索（Vector + BM25）
- CrossEncoder重排序
- 智能分块
- 查询优化

**适合**：希望优化检索效果的开发者

**预计时间**：3-4小时

---

- 自主规划

**适合**：想学习Agent开发的学习者

**预计时间**：4-5小时
